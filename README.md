# Mein dotfiles

A tidy `$HOME` is a tidy mind.

## Quick start

```sh
# Make sure your partitions are set up by this point
git clone https://github.com/hlissner/dotfiles /etc/dotfiles
/etc/dotfiles/deploy HOSTNAME   # installs nixos to /mnt
```
